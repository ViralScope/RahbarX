# RahbarX - Windows Performance Optimizer

A comprehensive PowerShell utility to optimize Windows 10/11 for gaming and maximum performance.

## Features

- **Game Mode Optimization** - Disable unnecessary services for gaming
- **Network Optimization** - TCP/IP tweaks for lower latency
- **Windows Debloater** - Remove pre-installed bloatware
- **VBS/HVCI Disable** - Disable virtualization-based security for performance
- **Visual Effects** - Disable Windows animations and effects
- **Power Plan** - Apply Ultimate Performance power plan
- **Memory Optimization** - Clean standby memory and optimize paging
- **Complete Rollback** - Undo all changes with one click

## Requirements

- Windows 10 (Build 19041+) or Windows 11
- PowerShell 5.1 or later
- Administrator privileges

## Usage

1. Download `RahbarX.ps1`
2. Right-click and select "Run with PowerShell"
3. Confirm administrator elevation when prompted
4. Use the GUI to apply desired optimizations

## Safety Features

- Creates System Restore point before changes
- Backs up registry and service states
- Domain-joined computer detection
- Network connection safety checks
- Complete rollback functionality
